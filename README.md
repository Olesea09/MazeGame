```
# Maze Game 
Projet fait par : Hiba Bacha, Olesea Popa et Rafia Ben Slama.

## Instructions pour Compiler

1. Décompressez l'archive ZIP contenant les codes sources.
2. Ouvrez le projet dans IntelliJ IDEA ou un autre IDE compatible avec Gradle.
3. Synchronisez Gradle si nécessaire.
4. Compilez le projet avec la commande suivante dans le terminal intégré de l'IDE :
   ./gradlew build

## Instructions pour Exécuter

1. Lancez le jeu en exécutant la commande suivante dans le terminal :
   ./gradlew :lwjgl3:run
2. Une fenêtre s'ouvre et le jeu démarre.

```
